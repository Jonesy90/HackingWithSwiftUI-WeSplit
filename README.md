# Hacking With SwiftUI - WeSplit

This is Project 1 of the Hacking With SwiftUI learning journey. The end result of this app is to have an check-sharing app that calculates how to split a bill based on the number of people and how much to tip on top of the final bill.

This project introduced and began to solidify how Swift and SwiftUI work together.

## What I Learned
1. @State Properties
2. @FocusState Properties
3. NavigationStack
4. Form
5. Button
6. Picker
7. TextField
8. ForEach
